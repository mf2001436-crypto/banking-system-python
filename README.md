# Banking System (Python)

A console-based Banking System developed using Python. This project allows users to perform basic banking operations through a simple menu-driven interface.

## Features

- Deposit Money
- Withdraw Money
- Check Account Balance
- Menu-Based User Interface
- Function-Based Program Structure

## Technologies Used

- Python

## How to Run

1. Download or clone the repository.
2. Open the project in VS Code.
3. Run the Python file:

```bash
python banking_system.py
```
